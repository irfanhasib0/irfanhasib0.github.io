1. Overview :


Code for Multi Layer Neural Network is implemented here from scratch, using raw Python programming language. Object oriented programming is used to better organize the code and making it user frendly. A Layer and a Network Object is the backbone of the implementation. A stack of Layer objects can be used to build a Network. Network object has the following fucntions(1) Layer :: "forward_propagation", (2) Layer :: "backward_propagation", (3) Layer :: "update_weights" . I have tested it with Banknote dataset. It was fitting good. Result can be found in Results section
